# Plugin API Reference

Reference for creating custom plugins and extensions.

## Plugin Types

1. **Protocol Plugins** - REST, GraphQL, gRPC, WebSocket, etc.
2. **Format Plugins** - JSON, XML, CSV, Protocol Buffers, etc.
3. **Authentication Plugins** - Custom auth providers

## Creating Protocol Plugins

See [Custom Plugins Guide](../guides/advanced/custom-plugins.md) for detailed examples.

## Plugin Registration

Plugins can be registered via ServiceLoader or programmatically.
