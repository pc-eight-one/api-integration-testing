## Installation

See [Installation Guide](./installation.md) for detailed instructions.

## Quick Start

1. Install the plugin from JetBrains Marketplace
2. Open a project with API Integration Testing Framework
3. Right-click on a test file and select "Run API Tests"
4. View results in the Test Explorer panel

## System Requirements

- IntelliJ IDEA 2023.1 or newer
- Kotlin Plugin enabled
- JDK 11 or higher
